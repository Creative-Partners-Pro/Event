GitHub Synchronization

Настройте интеграцию, чтобы сохранять изменения напрямую в ваш репозиторий на GitHub.

Token must have 'repo' scope. Don't share this token with anyone.

Как получить токен?

  1. В GitHub: Settings (профиля) -> Developer Settings -> Personal Access Tokens -> Tokens (classic)
  2. Нажмите "Generate new token (classic)"
  3. Назовите его (например, "Menu Admin") и отметьте галочку repo
  4. Скопируйте токен и вставьте его сюда.
